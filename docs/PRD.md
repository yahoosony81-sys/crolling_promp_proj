# TrendScrape Prompt (가칭)

**주간 트렌드 반영 크롤링·스크래핑 + 매주 업데이트되는 실행형 프롬프트 제공 서비스**

---

## 1. Overview

### Problem Statement

* **문제 정의**
  온라인에서 돈이 되거나 성과로 이어지는 정보는 항상 “트렌드”를 중심으로 움직이지만,
  대부분의 사람들은

  1. 지금 무엇이 핫한지 알기 어렵고
  2. 정보를 직접 수집·정리하는 데 시간이 많이 들며
  3. 수집한 정보를 실제 결과물(분석, 콘텐츠, 수익)로 전환하지 못한다.

* **왜 중요한가**

  * 트렌드는 빠르게 변하며, **1~2주만 늦어도 기회는 사라진다**.
  * 특히 콘텐츠 제작자(릴스·숏츠), 온라인 셀러, 투잡을 원하는 개인에게
    “정보 수집 속도 + 실행 속도”는 곧 경쟁력이다.
  * 기존 프롬프트 서비스는 **항상 같은 프롬프트**를 제공해
    실제 트렌드 반영이 어렵다.

* **타겟 고객**

  * 연령: **18세 ~ 60세**
  * 주요 사용자:

    * 주식·부동산·코인 등 투자 관심층
    * 스마트스토어·쿠팡 등 온라인 셀러
    * 네이버 블로그·콘텐츠 제작자
    * **인스타 릴스 / 유튜브 숏츠 제작자**
    * **투잡·부업이 필요한 직장인 및 프리랜서**
  * 공통 니즈:
    👉 “지금 뜨는 주제를 빠르게 얻고, 바로 실행해서 성과를 만들고 싶다”

* **전략적 적합성**
  본 서비스는 “AI와 자동화를 통해 정보 활용의 격차를 줄인다”는 비전과 부합하며,
  단순 프롬프트 제공이 아닌
  **트렌드 기반 실행 도구**라는 차별적 포지션을 가진다.

* **핵심 전제 / 가정**

  * 프롬프트의 가치는 “정교함”보다 **시의성(트렌드 반영)** 에서 크게 갈린다.
  * 사용자는 무료 체험 후 가치가 명확하면 유료 전환한다.
  * **‘매주 업데이트되는 프롬프트’**는 기존 프롬프트 서비스와의 핵심 차별점이다.

## 2. Proposed Work

### 솔루션 개요

사용자는 무료 프롬프트로 서비스를 체험하고,
**매주 트렌드를 반영해 새로 업데이트되는 크롤링 데이터 + 프롬프트 패키지**를
월 9,900원 구독으로 지속적으로 활용한다.

---

### 핵심 기능 (MVP)

1. **무료 프롬프트 템플릿 10선**

   * 트렌드 요약, 블로그 작성, 상품 분석, 콘텐츠 아이디어 등
2. **주간 트렌드 큐레이션 (유료)**

   * 매주 업데이트되는 분야별 트렌드 주제
3. **배치 기반 크롤링·스크래핑 (유료)**

   * 주 1~2회 자동 수집 및 요약
4. **주간 트렌드 패키지**

   * “이번 주 트렌드 + 요약 데이터 + 프롬프트” 묶음 제공
5. **목적별 프롬프트 제공**

   * 분석 / 콘텐츠 제작 / 판매 / 투자 등
